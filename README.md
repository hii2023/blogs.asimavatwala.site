# blogs.asimavatwala.site

Personal blog platform for **Altaf Simavatwala** at [asimavatwala.site](https://asimavatwala.site), built with a modern Next.js stack.

## What It Does

A full-featured blog site with an in-browser rich text editor, allowing Altaf to write, publish, and manage articles without a third-party CMS.

## Key Features

- **TipTap rich text editor** — browser-based writing with Starter Kit, Link, Underline, and Placeholder extensions
- **MDX / Markdown blog posts** — stored in the `posts/` directory
- **TypeScript throughout** — fully typed codebase
- **Tailwind CSS styling** — clean, responsive design
- **Static asset hosting** via `public/` directory
- **Next.js App Router** — fast page loads and SEO-ready

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Next.js 14 | Framework & routing |
| React 18 | UI components |
| TypeScript | Type safety |
| Tailwind CSS | Styling |
| TipTap | Rich text editor |
| PostCSS | CSS processing |

## Project Structure

```
src/          -> App source code
posts/        -> Blog post content (MDX)
public/       -> Static assets
```
